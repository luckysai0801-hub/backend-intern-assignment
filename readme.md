# Backend Developer (Intern) Assignment

This project implements a scalable REST API with secure authentication, role-based access control, and CRUD operations, along with a simple frontend UI for interacting with the APIs.

## Features
- User registration and login with JWT authentication
- Role-based access (user/admin)
- CRUD operations for tasks
- Secure password hashing
- Input validation and error handling
- API documentation using Swagger
- Simple frontend for API interaction

## Tech Stack
- Backend: Python, FastAPI, SQLAlchemy
- Frontend: HTML, CSS, JavaScript
- Database: SQLite (easily scalable to PostgreSQL)

## Setup Instructions
1. Install dependencies
2. Run backend server
3. Open frontend
